### Для запуска приложения в первый раз, необходимо выполнить следующие шаги:

1. Установить зависимости `npm install`
2. Создать в корне проекта файл `.env` и добавить туда строку `DATABASE_URL="postgresql://task_user:task_password@localhost:5432/task_manager_db"`
3. Необходимо запустить `Docker Desktop` приложение
   1. У вас должен быть установлен докер(https://www.docker.com/products/docker-desktop/)
   2. Если у вас нет образа для данного приложения, то в корне проект в терминале выполните данную команду: `docker-compose up -d`
4. Необходимо сгенерировать типы данных от prisma, для этого в консоли выполняем команду `npx prisma generate`, данную команду необходимо выполнять каждый раз при изменении схемы базы данных в файле `prisma/schema.prisma`
5. Необходимо применить миграции к базе данных, которые были ранее созданы, для этого в консоли выполните команду `npx prisma migrate dev`
6. Запускаем приложение командой `npm run start:dev`
